# ⌨️ Keyboard Visualizer

An interactive virtual keyboard that visually responds to user input. Pressing keys highlights them on screen, showing which finger should be used. Features smooth animations for key presses and supports Backspace, Shift, and CapsLock.

## Features
- Interactive visual keyboard
- Highlights keys based on finger placement
- Animations for key presses
- Handles special keys: Backspace, Shift, CapsLock
- Fully responsive and visually engaging

## Tech Stack
- HTML5
- CSS3
- JavaScript
