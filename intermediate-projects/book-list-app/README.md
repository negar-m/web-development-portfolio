A simple interactive Book List web application built with HTML, CSS, and JavaScript. Users can add books with title, author, and year, delete books, and clear all books. All data is saved in localStorage so it persists after refreshing the page.

- Add new books with title, author, and year
- Delete individual books
- Clear all books
- Data persists with localStorage
- Interactive and responsive UI
