# Text Generator ✍️

A simple text generator that allows users to generate multiple paragraphs of dummy text instantly by specifying the number of paragraphs.

## Features
- Generate multiple paragraphs of dummy text
- User-specified number of paragraphs
- Instant display of generated text
- Clean and responsive UI with Bootstrap 4
- Simple form handling with JavaScript

## Tech Stack
- HTML5
- CSS3
- Bootstrap 4
- JavaScript
- DOM Manipulation
