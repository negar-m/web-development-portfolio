# KeyCode Viewer App

A simple and interactive JavaScript application that displays keyboard event information when the user presses any key.

## Overview

This project is a front-end utility app built with vanilla JavaScript, HTML, and CSS.  
It listens for keyboard events and visually displays useful properties of the pressed key, making it helpful for learning and debugging JavaScript keyboard events.

## Features

- Detects keyboard input in real time
- Displays key event details:
  - `event.key`
  - `event.keyCode`
  - `event.which`
  - `event.code`
  - `event.location`
- Clean and minimal UI
- Beginner-friendly and educational
- No external libraries required

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6)
  - DOM Manipulation
  - Event Handling
